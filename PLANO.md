# Banco Imobiliário

## Objetivo

Recriar o jogo Banco Imobiliário para navegador utilizando HTML, CSS e JavaScript.

O objetivo inicial é reproduzir fielmente as regras e a experiência do jogo clássico.

Após a conclusão da versão 1.0, o projeto evoluirá para uma versão própria, com melhorias visuais, novas mecânicas e identidade original.

---

## Tecnologias

- HTML
- CSS
- JavaScript
